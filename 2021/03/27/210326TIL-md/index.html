Tree

특징

  1. 그래프의 특수한 형태
  2. 모든 노드는 서로 연결되어 있다.(따로 떨어져 외딴섬에 있는 노드는 없음)
  3. 방향성을 가지고 있다.
  4. 하나의 노드에서 다른 노드로 이동하는 경로는 유일핟.
  5. Acyclic하다. (비순환)

용어

  1. root node: 최상위 노드 (가장 낮은 level에 있는 Node)
  2. child node: 자식 노드
  3. parent node: 부모 노드
  4. leaf node(입새 노드): 자식 노드가 없는 노드
  5. Internal node(내부노드): leaf node 외의 노드
  6. edge: 연결선 (node의 수 - 1)
  7. path(경로): 두 노드를 연결하는 에지의 시퀀스
  8. level, depth: 루트 노드로부터 경로의 길이
  9. height: 트리의 높이로 가장 큰 레벨 값

종류

  • 정 이진 트리(Full Binary Tree, Perfect Binary Tree): leaf노드를 제외하고 모든 노드가 자식을 2개 가지고 있다.

  • 완전 이진 트리(Complete Binary Tree): 왼쪽부터 순서대로 채워 나가는 트리

  • 균형 이진 트리(Balanced Binary Tree):

  1. skewed Tree: 한쪽 방향으로만 쭉 이어지는 트리
  2. Balanced Binary Tree: 임의의 규칙을 가지고 균형을 이루고 있는 트리

순회

  • 깊이 우선순회(preorder, Depth-First Traversal): node -> left -> right
  • 대칭 순회(Inorder, Symmetric Traversal): left -> node -> right
  • 후위 순회(Postorder): left -> right -> node

탐색

  • 너비 우선탐색(Breath-First-search; BFS): 낮은 레벨부터 왼쪽부터 탐색

  • 깊이 우선탐색(Depth-First-search; DFS):
    node -> left -> right 순으로 탐색(left 자식 노드쪽으로 깊어지며 탐색함)
